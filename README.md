# Excel Sales Analytics

## Overview
This project focuses on analyzing sales data using Microsoft Excel to evaluate customer performance and market trends. The analysis helps identify high-performing markets, sales gaps, and opportunities for improvement.

## Tools Used
- Microsoft Excel (Advanced)
- Pivot Tables
- Charts & Visual Analysis
- Conditional Formatting
- Business KPIs
